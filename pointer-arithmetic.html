What is Pointer Arithmetic?

Pointer arithmetic allows programmers to perform arithmetic operations on addresses. It can simplify tasks like iterating through arrays.

Basic Operations:

  • Increment (++): Moves the pointer to the next element of the type it points to.
  • Decrement (--): Moves the pointer to the previous element of the type it points to.
  • Addition (+): Adds an integer to the pointer, moving it forward in the array.
  • Subtraction (-): Subtracts an integer from the pointer, moving it backward in the array.

Example:


#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *p = arr; // Pointer to the first element of the array

    for (int i = 0; i < 5; i++) {
        printf("Value: %d, Address: %p\n", *p, (void*)p);
        p++; // Move to the next integer
    }
    return 0;
}

In the example above, we increment the pointer p to access each element in the array arr.
