Understanding Arrays and Pointers

In C, an array is a collection of elements of the same type stored in contiguous memory locations. Pointers can be used to point to the first element of an array.

Example of Array and Pointer Usage:


#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *p = arr; // Pointer to the first element of the array

    for (int i = 0; i < 5; i++) {
        printf("Value at arr[%d] = %d\n", i, *(p + i)); // Accessing array using pointer
    }
    return 0;
}
