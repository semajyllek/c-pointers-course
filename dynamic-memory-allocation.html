Understanding Dynamic Memory Allocation

Dynamic memory allocation allows you to allocate memory at runtime, providing flexibility in memory management.

Common Functions

  • malloc: Allocates a specified number of bytes.
  • calloc: Allocates space for an array of elements and initializes them to zero.
  • realloc: Resizes previously allocated memory.
  • free: Frees previously allocated memory.

Example: Dynamic Array Allocation


#include <stdio.h>
#include <stlib.h>

int main() {
    int n;
    printf("Enter the number of elements: ");
    scanf("%d", &n);
    
    // Allocate memory for n integers
    int *arr = (int *)malloc(n * sizeof(int));
    
    if (arr == NULL) {
        printf("Memory allocation failed!\n");
        return 1; // Exit if allocation failed
    }

    // Initialize and display the array
    for (int i = 0; i < n; i++) {
        arr[i] = i + 1; // Initialize with values
        printf("%d ", arr[i]);
    }
    
    // Free the allocated memory
    free(arr);
    return 0;
}

Best Practices

Always check if the memory allocation was successful. Additionally, remember to free any dynamically allocated memory to prevent memory leaks.
