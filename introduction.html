What are Pointers?

Pointers are variables that store the memory address of another variable. They are a powerful feature of the C language.

Basic Pointer Syntax


int main() {
    int x = 10;        // normal integer variable
    int *p;           // pointer variable
    p = &x;           // storing the address of x
    printf("%d", *p); // dereferencing pointer to get value of x
}

In this example, p is a pointer to an integer variable x. The expression &x gets the memory address of x, which is then stored in p.
